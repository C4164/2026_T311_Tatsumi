# ゲーム制作演習Ⅲ企画

制作するもの：スロット（ボーナスタイプ）

開発環境：C++

使用するライブラリ：DirexrX

詳細：今回はATではなくシンプルなリアルボーナスタイプのスロットを作成。ボーナス抽選をモードで管理し、ボーナスの種類はBigBounusとRegularBounusの二種類とする。モードは低確、通常、高確の三つでそれぞれのモードに応じてボーナスの抽選を行う。

目標：今回の目的はスロットの根幹となる部分の作成を目標にしたい。まずはスロットの基本的な仕組みを学びたいので、とりあえずレバーオンで小役を抽選し、抽選された小役に応じてリール制御させて目に応じて払出しを計算するという流れをこのゲーム制作で完成させられたらと思う。
